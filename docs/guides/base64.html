Guide

Base64 turns bytes into text. Base64URL makes that text URL-safe.

Use Base64 when a binary value must travel through text-only storage or protocols. Use Base64URL when the encoded value will appear in URLs, filenames, tokens, or other places where `+`, `/`, and padding are awkward.

Base64

RFC 4648 Base64 uses `A-Z`, `a-z`, `0-9`, `+`, `/`, and `=` padding. AnoSecureKit decoders require canonical input.

Base64URL

Base64URL replaces `+` with `-`, `/` with `_`, and AnoSecureKit emits it without padding.

Not encryption

Base64 is encoding, not security. Anyone can decode it. Do not use it to hide secrets.

When to use it

  • Use Base64 for binary data in text formats that expect the standard alphabet.
  • Use Base64URL for compact URL-safe tokens and identifiers.
  • Do not use either as encryption, authentication, compression, or password protection.

CLI example

AnoSecureKit exposes Base64URL through the CLI. It writes decoded bytes to stdout, so automation should treat output as bytes. 

anosecurekit base64url-encode --text abc
anosecurekit base64url-decode --text YWJj

C++ API example

#include "anosecurekit/base64.hpp"

#include <cstddef>
#include <string>

anosecurekit::bytes data{std::byte{'f'}, std::byte{'o'}, std::byte{'o'}};

std::string standard = anosecurekit::base64_encode(data);    // "Zm9v"
std::string url_safe = anosecurekit::base64url_encode(data); // "Zm9v"

anosecurekit::bytes decoded = anosecurekit::base64url_decode(url_safe);

Validation behavior

AnoSecureKit decoders are strict. Standard Base64 rejects malformed padding, whitespace, and Base64URL-only characters. Base64URL rejects standard `+` and `/`, rejects padding, and requires canonical input.

Malformed input raises `anosecurekit::error` with `anosecurekit::error_code::invalid_encoding`.

Security notes

  • Base64 does not provide confidentiality or integrity.
  • Encoding a key, password, token, or packet does not make it safe to log or publish.
  • Use HMAC or AEAD when you need authenticity or tamper detection.
  • Use AnoSecureKit file or packet encryption when you need encryption.
