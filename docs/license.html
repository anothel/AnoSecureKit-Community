License

MPL-2.0, with a file-level boundary.

AnoSecureKit allows commercial and closed-source application use while keeping distributed modifications to AnoSecureKit-covered files open under MPL-2.0.

Allowed use

Commercial use is allowed. Applications may use AnoSecureKit as a separate library without forcing unrelated application files open because of AnoSecureKit alone.

Modified files

If modified AnoSecureKit source files are distributed, those modified AnoSecureKit-covered files should remain available under MPL-2.0.

Separate product boundary

AnoSecureKit Enterprise and AnoCrypto-C are separate repositories and products. Community includes neither proprietary source nor an AnoCrypto-C adapter, and using separately licensed products does not change the MPL-2.0 license of Community files.
