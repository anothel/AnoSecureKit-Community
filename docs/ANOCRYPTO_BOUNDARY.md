# AnoCrypto-C Boundary

AnoCrypto-C is a separate external project. It is not included in, consumed by,
or shipped from AnoSecureKit Community.

Community may contain a generic private provider contract and a build-tree-only
external provider seam. That seam does not make AnoCrypto-C a Community backend
and does not permit silent fallback.

The canonical boundary is `docs/CROSS_REPOSITORY_BOUNDARY.md`.
