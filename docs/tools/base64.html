Browser-local tool

Encode and decode Base64 text in your browser.

This tool runs on the page only. It does not upload input, output, or errors to a server.

Ready.

Rules

  • Encode treats input as UTF-8 text.
  • Decode expects canonical Base64 or unpadded Base64URL.
  • Decoded bytes must be valid UTF-8 text to display here.
  • Base64 is encoding, not encryption.
