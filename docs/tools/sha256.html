Browser-local tool

Calculate SHA-256 for UTF-8 text in your browser.

This tool uses Web Crypto in the browser. It does not upload input, output, or errors to a server.

Ready.

Rules

  • Input is encoded as UTF-8 text before hashing.
  • The output is a 32-byte SHA-256 digest shown as 64 lowercase hex characters.
  • SHA-256 is not encryption and does not hide the input.
  • Do not use raw SHA-256 digests for password storage.
