Browser-local demo

Encrypt and decrypt UTF-8 text with AES-256-GCM in your browser.

This educational demo uses Web Crypto in the browser. It does not upload keys, nonces, plaintext, ciphertext, tags, AAD, or errors to a server.

Ready.

Rules

  • This is a raw AES-256-GCM text demo, not an AnoSecureKit `SKT1` packet encoder.
  • Key must be 32 bytes as 64 hex characters.
  • Nonce must be 12 bytes as 24 hex characters. Never reuse a nonce with the same key.
  • AAD is authenticated but not encrypted. Decrypt requires the exact same AAD.
  • Do not paste production secrets into browsers or devices you do not trust.
