Browser-local tool

Generate URL-safe random tokens in your browser.

This tool uses `crypto.getRandomValues()` in the browser. It does not upload or store generated tokens.

Ready.

Rules

  • The byte size is the number of random bytes before Base64URL encoding.
  • Output uses `A-Z`, `a-z`, `0-9`, `-`, and `_` with no `=` padding.
  • Generated tokens are not stored by this page.
  • Treat generated tokens as secrets if possession grants access.
