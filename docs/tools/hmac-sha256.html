Browser-local tool

Calculate HMAC-SHA-256 for UTF-8 text in your browser.

This tool uses Web Crypto in the browser. It does not upload keys, input, output, or errors to a server.

Ready.

Rules

  • Message input is encoded as UTF-8 text before signing.
  • Text keys are encoded as UTF-8. Hex keys must contain an even number of hex characters.
  • The output is a 32-byte HMAC-SHA-256 tag shown as 64 lowercase hex characters.
  • HMAC detects tampering but does not encrypt or hide the message.
  • Do not paste production secrets into browsers or devices you do not trust.
