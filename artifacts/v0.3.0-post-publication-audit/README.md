# AnoSecureKit Community v0.3.0 Historical Evidence

Status: HISTORICAL

The previous repository snapshot retained expanded release archives and binaries
for the v0.3.0 post-publication audit. Those files are no longer kept in the
current source tree because they duplicate published release assets and make
source packaging unnecessarily large.

GitHub Release. They do not prove the state of v0.4.0 or any later revision.
